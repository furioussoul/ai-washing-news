## ⚙️ 02 极简多智能体流转图（核心架构）

为了教学效果，我们将复杂的 BettaFish 简化为经典的 **“数据采集 -> 分析决策 -> 报告输出”** 三步走模型。把它映射到我们的【AI TEAM】产品逻辑中：

*   **👤 用户输入**：输入任何热点事件（例如：“特斯拉降价”、“苹果新品发布”）。
    *   ⬇️ 触发流转
*   **🕵️‍♂️ 【AI TEAM - 数据侦察兵 Spider】**：
    *   **任务**：模拟全网搜索，抓取相关的社交媒体评论、新闻标题。（教学中用大模型模拟生成高逼真度的数据，保证演示 100% 成功且瞬间出结果）。
    *   ⬇️ 传递 JSON 数据
*   **🧠 【AI TEAM - 情绪分析师 Analyst】**：
    *   **任务**：读取上一步的数据，判断每条评论的情感倾向（正向、负向、中立），提取核心风险词，计算“舆情危机指数”。
    *   ⬇️ 传递分析指标
*   **✍️ 【AI TEAM - 报告撰写专家 Writer】**：
    *   **任务**：将冰冷的指标转化为 CEO 级别的可视化看板（Dashboard），包含 Echarts 图表、风险预警灯、以及一段高管总结摘要。

## 🎭 03 【AI TEAM】底层角色设定 (Role Prompts)

这部分是向“有点技术基础”或“程序员”展示肌肉的地方。告诉他们：在真实的【AI TEAM】企业版中，我们是这样设定每个 Agent 的身份的。

### Agent 1: 舆情侦察兵 (Scout)
```text
Role: 你是一个资深的全网数据侦察兵。
Task: 给定一个[热点事件]，你需要立即生成10条极具代表性的全网评论。必须包含：平台(微博/推特/小红书)、时间戳、评论内容。
Constraint: 评论内容必须真实还原网民语气，包含阴阳怪气、极度赞美、中立客观等不同维度。输出格式必须为纯净的JSON数组。
```

### Agent 2: 情绪分析师 (Analyst)
```text
Role: 你是一位华尔街级别的舆情风险分析师。
Task: 接收[侦察兵的JSON数据]，对每条数据进行情感极性打分(-10到10)。提取出TOP 3出现频率最高的[风险关键词]。
Output: 计算出整体的"舆情危机指数(0-100)"，并给出一段20字的简短定性分析。
```

### Agent 3: 报告撰写专家 (Writer)
```text
Role: 你是顶级的商业BI报告撰写专家。
Task: 接收[分析师的评估结果]，将数据转化为结构化的执行摘要（Executive Summary）。语言风格必须专业、克制、且具有洞察力，适合直接呈报给CEO阅读。
```

## 🚀 04 Vibe Coding 神级演示 Prompt

> **讲师操作指南**：在直播或录制视频时，打开 Cursor、Windsurf 或 Bolt.new 等自然语言编程工具。直接新建一个 React/Vite 项目，把下面这段长 Prompt 复制进去，一键回车！即可跑出惊艳全场的系统。

### 🌟 神级提示词（直接复制使用）

```text
请帮我用 React + Tailwind CSS + Recharts (或 Echarts) 开发一个「BettaFish AI 舆情分析系统」的极简复刻版页面。
这不仅是一个 UI，请你在前端代码中模拟一个【多智能体 (AI TEAM)】的协作流程：

1. 页面分为左右两栏。左侧是“控制台”，右侧是“CEO 数据看板”。
2. 左侧控制台：
   - 有一个炫酷的输入框，默认 placeholder 是“输入热点事件，例如：最新款AI手机发布”。
   - 一个“启动 AI TEAM”按钮。点击后，展示多智能体的工作流动画。
3. 模拟多智能体工作流（使用 setTimeout 模拟，并在界面上用终端风格的日志打印出来）：
   - [🕵️ 数据侦察兵] 正在全网抓取 10,245 条社交媒体评论... (1秒后完成)
   - [🧠 情绪分析师] 正在通过 NLP 模型进行情感极性打分与风险词提取... (1.5秒后完成)
   - [✍️ 报告撰写专家] 正在生成 CEO 执行摘要与可视化报表... (1秒后完成)
4. 右侧 CEO 数据看板（工作流完成后显示，带有淡入动画）：
   - 顶部指标卡：舆情危机指数（例如：78/100 红色预警）、声量总数、情感倾向分布（笑脸/哭脸比例）。
   - 中间图表：用 Recharts 渲染一个美观的饼图（正向30%，负向55%，中立15%）和一个风险关键词的列表标签。
   - 底部报告区：显示一段由“报告撰写专家”输出的专业执行摘要：“当前舆情呈现高风险态势，负面情绪主要集中在‘电池续航’与‘发热’问题上，建议公关部立即介入处理。”
5. 整体风格要求：极客风、暗黑模式 (Dark Mode)、类似硅谷顶级 SaaS 产品的质感，使用毛玻璃特效 (Glassmorphism)。请在一个文件中完成所有组件，确保我一键运行就能看到完美效果。
```
